# Cadastro de Pessoas

Sistema simples de cadastro de pessoas desenvolvido em Java como parte dos meus estudos em Programação Orientada a Objetos (POO).

O projeto permite cadastrar, listar e remover pessoas através de um menu interativo executado no terminal.

## Funcionalidades

- Cadastrar pessoas
- Listar todas as pessoas cadastradas
- Remover pessoas pelo nome
- Menu interativo no console
- Validação para lista vazia
- Mensagens de feedback para o usuário

## Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)
- ArrayList
- Scanner
- Laços de repetição
- Estruturas condicionais

## Estrutura do projeto


## Como executar

1. Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/cadastro-de-pessoas.git
```

2. Abra o projeto em sua IDE de preferência (IntelliJ IDEA, Eclipse ou VS Code).

3. Execute a classe `Main.java`.

## Exemplo de utilização

```
==============================
      CADASTRO DE PESSOAS
==============================
(1) Cadastrar pessoa
(2) Listar pessoas
(3) Remover pessoa
(4) Sair

Escolha uma opção: 1

Digite o nome: Luan

Pessoa cadastrada com sucesso!
```
